SyndicateRoom

Designing scalable investment journeys with UX + Frontend development

At SyndicateRoom, I led both design and development, delivering modular dashboards and investment flows for thousands of investors. By integrating user research, journey mapping, reusable UX/UI patterns, and production-ready frontend components, we created a system that supported rapid growth and improved retention, taking the fund from zero to £77M deployed.

← Back

Problem

  • No structured design system.
  • Investment process was inconsistent and hard to navigate.
  • Needed scalable flows to support rapid growth.
  • Legacy monolithic codebase.

Process

  • Research: Stakeholder interviews, personas, journey mapping.
  • Design: Wireframes, prototypes, iterative usability testing.
  • System: Modular dashboards, investment flow patterns, component library.
  • Development: Built React/TypeScript modular components and templates on a microservice foundation for performance and maintainability.
  • Team Growth: Expanded from 2 → 6, mentoring designers & developers.

Solution

  • Designed dashboards for 2,000+ investors using a modular, reusable frontend system.
  • Created a component library to standardize UI and reduce dev time for new flows.
  • Implemented a structured mentorship program to accelerate developer upskilling.

Impact

  • Grew fund from scratch to £77M deployed.
  • Achieved 76% customer retention.
  • Reduced development time for new flows by standardizing reusable components, scaling team efficiency.
