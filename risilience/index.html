Designing accessible dashboards for climate risk data with frontend scalability

At Risilience, I led the design and frontend teams to create accessible, user-friendly dashboards for enterprise clients. Combining UX research with scalable frontend architecture, we transformed dense datasets into intuitive interfaces that improved usability, accessibility, and customer satisfaction across the platform.

← Back

Problem

  • Dashboards were difficult to navigate.
  • Accessibility compliance was lacking.
  • No design system or scalable UI components existed.

Process

  • User Research: Stakeholder workshops, client interviews, personas.
  • Prototyping: Wireframes, journey maps, iterative usability testing.
  • Design System: Component library, style guide, React/TypeScript dashboards, cross-platform consistency.
  • Accessibility: WCAG 2.2 compliance built into every release.

Solution

  • Redesigned dashboards with interactive charts and reusable component library (50+ elements).
  • Developed a frontend system for rapid updates, scalability, and maintainability.
  • Led team in visual storytelling, turning complex climate datasets into actionable insights.

Impact

  • +22% increase in customer self-service.
  • +46% increase in customer satisfaction.
  • Built and led design + frontend team from scratch, standardizing components and improving dev efficiency across the platform.
